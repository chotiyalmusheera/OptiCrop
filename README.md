# 🌱 OptiCrop - Smart Agricultural Recommendation System

## 📌 Project Description
This project uses Machine Learning to recommend the best crop based on soil nutrients and environmental conditions.

## 🚀 Inputs Used
- Nitrogen (N)
- Phosphorous (P)
- Potassium (K)
- Temperature
- Humidity
- pH Level
- Rainfall

## 🧠 Output
Predicts the best suitable crop (example: rice, maize, cotton, etc.)

## ⚙️ Technologies Used
- Python
- Flask
- Machine Learning
- Scikit-learn
- Pandas
- HTML, CSS, Bootstrap

## 🤖 Model Used
Random Forest Classifier

## 📊 Accuracy
98%+
